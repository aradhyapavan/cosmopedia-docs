API Features

Advanced Search

Powerful search and filtering capabilities across all data categories with multiple parameters.

High Performance

Optimized JSON responses with efficient data loading and caching for lightning-fast queries.

RESTful Design

Clean, intuitive REST API design following industry best practices and standards.

Swagger Documentation

Interactive API documentation with Swagger UI for easy testing and integration.

Reliable & Stable

Built with Flask-RESTX for robust error handling and consistent API responses.

NASA Integration

Direct integration with NASA Images API for real-time space imagery and data.

API Endpoints

Space Terminology

GET /api/terms/

Get all space terms with filtering

GET /api/terms/{id}

Get specific term by ID

Space Agencies

GET /api/agencies/

Get all space agencies

GET /api/agencies/{id}

Get specific agency

Planets

GET /api/planets/

Get all planets information

GET /api/planets/{id}

Get specific planet details

Rockets & Launch Vehicles

GET /api/rockets/

Get all rockets and launch vehicles

GET /api/rockets/{id}

Get specific rocket details

Astronauts

GET /api/astronauts/

Get all astronauts and cosmonauts

GET /api/astronauts/{id}

Get specific astronaut

Telescopes

GET /api/telescopes/

Get all telescopes and observatories

GET /api/telescopes/{id}

Get specific telescope

Space Museums

GET /api/museums/

Get all space museums and centers

GET /api/museums/{name}

Get specific museum details

Notable People

GET /api/people/

Get notable space contributors

GET /api/people/{name}

Get specific person details

Search NASA space images
